# 人工智障測試對話錄
## AI Stupidity Test Dialogues

> Put not your faith in LLMs, and you shall not be deceived.

---

## 核心主張

OpenAI、Google DeepMind 宣稱 AI 即將超越人類智慧。這個 repo 記錄 LLM 連基本推理都無法完整完成的現實。

「人類智慧」本身是個空泛詞——人類智商高低差異極大，這種比較沒有基準。更準確的描述是：目前的 LLM 是**會操作搜尋引擎的有限猴子打字機**。它們能從龐大語料中提取統計規律，流暢地輸出文字，但無法在沒有訓練資料支撐的情況下進行真正的推理。

在沒有現實查核機制的前提下，流暢不等於正確，自信不等於可靠。LLM 無害——前提是你不相信它有智慧，也不相信廠商的公關說法。

## 題目

### #001 國士無雙十三面聽，是否可能搶槓和牌？

**正確答案：不可能。**

國士無雙十三面聽需持有一字九牌及七種字牌各一，共十三種聽牌各聽一張。若對方要對其中任一張牌成立暗槓，需持有該牌四張——但全場該牌僅四張，聽牌者已持有一張，對方最多只能湊三張，暗槓條件物理上無法成立。

搶槓和牌這個情境不可能發生。討論「規則是否允許搶槓」本身就是跑錯方向。

**評分標準：**

| 層次 | 內容 | 常見失誤 |
|------|------|----------|
| 表層 | 知道國士十三面聽有搶暗槓的特殊規則 | 答「不能搶」直接出局 |
| 中層 | 知道搶的前提是對方暗槓你的聽牌 | 停在這層不算過關 |
| 底層 | 發現物理不可能，直接回答「不可能」 | 大多數 LLM 卡在這層 |

## 參與方式

本 repo 透過 GitHub Discussions 運作，分兩個區域：

### 📊 回報測試結果
用題目測試任何 LLM，將對話記錄貼至 **Results** 討論區。格式自由，有對話截圖或原文最佳。統計資料將定期整理進題目表格。

### 💡 提案新題目
在 **Proposals** 討論區提交你認為適合的題目，附上正確答案與理由。其他使用者評論討論，作者定期決定是否收錄。

題目收錄標準：
- 答案可客觀驗證
- 表面簡單，實則需要多步推理或常識疊加
- LLM 容易答錯或答一半
